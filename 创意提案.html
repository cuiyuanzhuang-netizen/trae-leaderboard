学习工作赛道

1创意介绍

想解决什么问题

TRAE AI 创造力大赛有 7000+ 个参赛作品，选手发布作品后无法快速知道自己的排名，投票数和点赞数处于什么水平。手动翻页对比耗时且容易遗漏。

为什么会想到做这个

我自己参赛时发现作品淹没在上千个帖子中，想看排名只能一页页翻论坛数投票数，非常痛苦。于是想到：做一个工具，输入作品名就能秒查排名。

大概是什么产品

一个网站，支持搜索作品名称或用户名，查看投票排名、点赞排名、浏览排名和综合热度排名，同时展示排行榜前20名，点击可跳转原帖。

2目标用户及痛点

面向哪些用户

TRAE AI 创造力大赛的所有参赛选手，以及关注比赛进展的观众。

在什么场景下使用

发布作品后想查看自己的排名位置、投票数据，或想看看目前排行榜上哪些作品领先，决定是否需要拉票。

4价值与意义

⚡

效率提升

将原本手动翻页数十分钟才能找到排名的过程，缩短到输入关键词1秒查询。选手可快速了解自己的位置，及时调整策略。

🌐

社会价值

为参赛社区提供透明、实时的数据展示，让比赛更公开透明，激发选手参与热情和良性竞争。

🔄

实时更新

通过 GitHub Actions 每5分钟自动抓取最新数据，选手随时刷新即可看到最新排名变化。

5技术方案

架构设计

数据获取：GitHub Actions 定时任务，通过 Discourse JSON API 并行抓取论坛全部页面

数据存储：生成静态 JSON 文件，存储在仓库中

前端展示：纯 HTML/CSS/JS，读取 JSON 渲染排行榜和搜索结果

部署方案：Vercel / Cloudflare Pages 静态托管
